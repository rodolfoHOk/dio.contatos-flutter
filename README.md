# Contatos Flutter App

> Desafio Criando um App Flutter de Lista de Contatos da DIO

## Tecnologias

- Dart
- Flutter
- back4app: Backend as a Service (BaaS)
- Rest API

### Bibliotecas adicionais

- flutter_dotenv
- dio
- image_picker
- image_gallery_saver
- image_cropper
